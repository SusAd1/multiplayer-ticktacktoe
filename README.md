# 🎮 Multiplayer Tic Tac Toe

A web-based Tic Tac Toe game built using **Flask (Python)** with **HTML, CSS, and JavaScript**.

## Features

- ✅ Local 2-player mode
- 🧠 Modular game logic using JavaScript class
- 🎨 Simple and responsive UI
- 🔁 Template system (Jinja2) for reusability
- 🔜 Multiplayer and AI modes coming soon

## How to Run

1. Make sure Python 3 and Flask are installed.
2. Open a terminal and run:

```bash
pip install flask
python app.py
